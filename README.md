# Aurevia One

**Concept & Product Demo by Dhanush**

Aurevia One is a frontend-only conceptual demonstration of a modern, AI-powered mutual fund investing platform. 

> **Note:** This project is a **demo** created for an exploratory internship/product discussion assignment. It does not use real stock data, does not have a real backend, and does not process real money or authentication. It is built to showcase product thinking, UX/UI problem-solving, and a scalable product vision.

## Purpose

The purpose of this demo is to:
- Reverse engineer and analyze existing fintech platforms (inspired by platforms like Groww and AssetsPlus).
- Simplify the investing experience for beginners.
- Integrate AI-guided investing concepts naturally into the user flow.
- Showcase a premium, startup-level presentation and modern UI aesthetics.

## Features Showcased
- **Why Aurevia One is Different:** A dedicated section explaining product differentiation against traditional platforms.
- **AI Assistant:** Simulated conversational AI for answering investment questions using plain language.
- **Goal-Based Dashboard:** A calm and confident dashboard focusing on goals rather than overwhelming metrics.
- **Explore Funds:** Curated fund discovery with AI confidence scores.
- **Learning Hub:** Contextual, easy-to-digest educational resources directly integrated into the platform.

## Tech Stack
- React
- Vite
- Tailwind CSS v4
- Framer Motion (Animations)
- Recharts (Data Visualization)
- Lucide React (Icons)
- React Router (Routing)

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Disclaimer
This is purely a frontend demonstration and concept. No real investments are made.
